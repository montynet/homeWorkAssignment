Morty App

Build the winners!

Purpose:

Help uncle Morty identify potential lotto picks based on his newspaper method.

Rules:

  • Each number(s) string needs to be separated by a comma
  • Numbers may be used more than once!
  • A number string is a potential pick if it's 7 unique numbers between 1 and 59. Digits must be used in order, and every digit must be used exactly once.
